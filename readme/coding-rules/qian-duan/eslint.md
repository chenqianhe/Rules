# ESLint

ESLint 是一个代码检查工具，用来检查你的代码是否符合指定的规范（例如： = 的前后必须有一个空格）。

* 规范
  * 行业推荐的规范； 在创建项目时，我们使用的是 JavaScript Standard Style 代码风格的规则
  * 自定义的规范。你和你的团队可以自行约定一套规范
* 使用ESLint的好处在于：多人协作时代码风格统一
